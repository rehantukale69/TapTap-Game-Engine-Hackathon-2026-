# TaPTaP Learning Game Engine

A modular web-based game engine designed to generate multiple educational games using JSON configuration.

## Features

- WebGL based rendering system
- Physics simulation using planck.js
- Audio system using Howler.js
- JSON-driven scene configuration
- Modular engine architecture

## Architecture

Main systems:

- Engine Manager
- Renderer
- UI System
- Simulation System
- Audio System
- Entity System
- Game State Manager
- Game Loader
- Engine Editor
