# 🎮 TapTap Engine – User Manual (How to Play)

---

## ▶️ Starting the Game

1. Run the project using a local server  
2. Open it in your browser  
3. The game will load automatically  

---

## 🎮 Game Controls

### 🕹️ Movement

- **← Left Arrow** → Move player left  
- **→ Right Arrow** → Move player right  

---

### 🧪 Sandbox Controls

- **A key** → Spawn boxes  
  - Boxes will fall and interact using physics  

---

### 💾 Utility Controls

- **B key** → Save current scene  

---

## 🎯 Gameplay (Endless Runner Mode)

- You control a player block  
- Obstacles move toward you  
- Avoid collisions  

---

### 💥 Game Over

- If the player hits an obstacle → game ends  
- Game Over screen appears  

---

### 🔁 Restart / Resume

Use on-screen buttons:

- **Resume** → Continue game  
- **Restart** → Start again  
- **Exit** → Return to menu  

---

## ⏸️ Pause System

- Pause menu can be accessed via UI  
- While paused:
  - Game stops  
  - UI remains interactive  

---

## 🔊 Audio Controls

- Use the **audio button (icon)** on screen  
- Click to:
  - 🔊 Turn sound ON  
  - 🔇 Turn sound OFF  

---

## 🧪 Sandbox Mode

- Press **A** to spawn boxes  
- Observe:
  - collisions  
  - stacking  
  - physics behavior  

> This mode demonstrates the engine's physics capabilities.

---

## 📊 Score System

- Score increases during gameplay  
- Displayed on screen  
- Tracks survival time / progress  

---

## 🧠 Tips

- Stay centered to avoid obstacles  
- Use smooth movement instead of rapid switching  
- In sandbox mode, try spawning multiple boxes quickly  

---

## ⚡ Summary

This prototype allows you to:

- Play a dodge-based runner game  
- Interact with UI systems  
- Toggle audio  
- Test physics in sandbox mode
